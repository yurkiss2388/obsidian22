`const` — ключевое слово языка для объявления констант. Константа — переменная, значение которой нельзя переназначить.

пример :
```js
const univer = {
	"история" : 5,
	"этика" : 4,
	"ОБЖ": 2
}

свойства обьекта можно переназначать:

univer['физика'] = 5;
univer.химия = 5;

для массивов :
const univerResault = [3,4,5];
univerResault[0] = 5;






```
